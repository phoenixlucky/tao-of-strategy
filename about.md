# 关于韬略之道

> **"天下同归而殊途，一致而百虑。"** ——《周易·系辞》

---

## 项目哲学

兵家与道家，看似水火不容——一个讲攻伐谋略、争胜天下；一个讲清静无为、超然物外。但细读原典会发现：**二者共用同一套底层逻辑**。

道家讲"反者道之动，弱者道之用"；兵家讲"以正合，以奇胜"、"避实击虚"。两家都在阐释"对立面的转化"，只是取向不同：

| 维度 | 道家（避世面） | 兵家（进取面） |
|------|---------------|---------------|
| 对"反"的态度 | 知止、守柔、归根 | 奇正相生、以奇胜 |
| 对"势"的态度 | 顺其自然、无为 | 任势、造势、因敌制胜 |
| 最终指向 | 逍遥、长生、归根 | 胜敌、存己、全胜 |

**范蠡**助越灭吴是兵家，三散千金是道家；**张良**运筹帷幄是兵家，从赤松子游是道家；**诸葛亮**淡泊明志是道家，六出祁山是兵家。

——他们不是两种人，是同一个人在不同境遇中的两面。

本项目的使命，就是把"一枚硬币的两面"展现在同一面墙上。

## 核心特色：一体两面标签

每条格言除标注"兵家/道家"归属外，额外赋予三面倾向标签：

### 进取面（兵家取向）

| 标签 | 含义 |
|------|------|
| 谋攻 | 谋略攻伐 |
| 奇正 | 奇正相生 |
| 速决 | 兵贵神速 |
| 形势 | 形与势的运用 |
| 诡道 | 兵者诡道 |
| 用间 | 用间使谍 |
| 任势 | 任势造势 |
| 庙算 | 庙堂谋算 |
| 全胜 | 不战而屈人之兵 |

### 避世面（道家取向）

| 标签 | 含义 |
|------|------|
| 无为 | 无为而治 |
| 不争 | 不争而善胜 |
| 守柔 | 柔弱胜刚强 |
| 知足 | 知足知止 |
| 逍遥 | 精神自由 |
| 归根 | 复归其根 |
| 玄同 | 玄妙齐同 |
| 坐忘 | 坐忘无己 |

### 转化面（融合——一体两面）

| 标签 | 含义 |
|------|------|
| 功成身退 | 功成弗居 |
| 大智若愚 | 大巧若拙 |
| 以退为进 | 退即为进 |
| 后发先至 | 后发优势 |
| 阴阳相生 | 对立统一 |

## 标签双打示例

同一句话，从不同面看有不同的解读：

> **老子"将欲歙之，必固张之"**
> - 在道家视野下是**微明**（幽深的智慧、洞察循环之理）
> - 在兵家视野下是**诡道**（示弱诱敌、欲擒故纵）
> - **标签**：`转化` `阴阳相生` `诡道`

> **孙子"不战而屈人之兵"**
> - 进取面——全胜战略，以最小代价获取最大胜利
> - 避世面——止戈为武，"兵者不祥之器，不得已而用之"
> - **标签**：`转化` `全胜` `不争`

> **诸葛亮"非淡泊无以明志，非宁静无以致远"**
> - 避世面——淡泊宁静，完全是道家修养功夫
> - 进取面——明志致远，却是进取的目标
> - **标签**：`转化` `修身` `无为` `庙算`

## 项目起源

"一体两面"的灵感来自中国哲学中一个最迷人但也最容易被忽略的现象：**在同一个人身上，兵家和道家可以并存，而且往往越是深刻的思想家/实践家，两者融合得越彻底。**

范蠡、张良、诸葛亮、王阳明……这些名字之所以穿越千年仍有魅力，正因为他们不是单纯的谋士或隐士，而是同时具备两种力量的人。

本项目试图通过格言这一最小单位，以"两面解读"为方法，重新呈现这种被割裂已久的整体性。
